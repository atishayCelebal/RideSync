# Project Tech Stack

**Backend:**

- Java
- Spring Boot
- Maven
- Docker Compose
- JUnit

**Frontend:**

- TypeScript
- React
- Vite
- npm / Yarn / PNPM
- Biome
- React Testing Library / Jest
- Playwright
- Mock Service Worker
- PWA / static assets

## Running the Backend

Inside the `backend` directory, run:

```
./mvnw spring-boot:run
```

## Running the Frontend

Inside the `frontend` directory, run:

```
pnpm dev
```
